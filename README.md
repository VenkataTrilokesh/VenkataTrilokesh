## 🚀 Featured Projects

### 🔬 [DataLens  Automated AI Data Scientist System](https://datalens-yourjrdatascientist.streamlit.app) &nbsp;![Live](https://img.shields.io/badge/Live-brightgreen)

> A fully automated **4-agent ML pipeline** that handles Data Understanding, EDA, Modelling, and Explainability  replacing a junior data scientist on any tabular CSV dataset.

- Trains **12–15 ML algorithms** per run (Ridge, Lasso, RF, XGBoost, LightGBM, SVM, ANN, etc.), selects top 3 by cross-validation score, and applies **RandomizedSearchCV** tuning  achieving **82–87% accuracy** on benchmark datasets
- Auto-detects class imbalance with **SMOTE oversampling**, data leakage, learning curve issues, and overfitting/underfitting diagnostics
- Outputs a **self-contained downloadable HTML report** with full SHAP explainability

`Python` `Streamlit` `Scikit-learn` `XGBoost` `LightGBM` `SHAP` `SMOTE` `Pandas` `Matplotlib` `Seaborn`
